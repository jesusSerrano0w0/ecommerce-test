# React + Vite

# este proyecto toma imagenes y productos de mercado libre 

# Stack Tecnológico
- **Framework:** React 19 + Vite
- **Routing:** React Router DOM v19
- **UI:** React Bootstrap + FontAwesome
- **Estado Global:** Context API + useContext
- **Datos:** Mock API simulada con latencia real services/api.js
- **Formato de precios:** Intl.NumberFormat COP - Colombia

# Instalación y Ejecución

- git clone https://github.com/jesusSerrano0w0/ecommerce-test.git
- cd ecommerce-test
- npm install
- npm run dev
